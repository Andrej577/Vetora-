# Dizajn

## Alat

Preporuka: Figma za UI dizajn web i mobilnih ekrana.

Alternativa za brzi prototip: Penpot, besplatan i open-source.

## Stil

- Bootstrap 4 grid i komponente
- Primarna boja: tamno zelena
- Sekundarna boja: topla zlatna za oznake i naglaske
- Font: sistemski font radi brzine i čitljivosti
- Kartice i paneli: 8px radius, lagana sjena, puno bijelog prostora

## Web ekrani

- Prijava osoblja
- Radna ploča
- Vlasnici
- Profil vlasnika
- Ljubimci
- Karton ljubimca
- Termini
- Računi
- Postavke ambulante

## Mobilni ekrani

- Današnji termini
- Brzo dodavanje vlasnika
- Brzo dodavanje ljubimca
- Karton ljubimca
- Kontakt vlasnika jednim dodirom

## MVP prioritet

1. Vlasnici
2. Ljubimci
3. Termini
4. Medicinski karton
5. Računi
6. Login i role
7. API za mobilnu aplikaciju ili vanjske integracije
